# SecondOrder (Website v1)

Modern, calm, dark-mode product website + playbook + toolkit for long-term investors.

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Pages
- `/` Home
- `/playbook` Playbook overview + sidebar
- `/playbook/asset-classes/*` Gold/Crypto/Equity/REITs
- `/toolkit` Free calculators (compound interest + EMI)
- `/pro` Pro teaser (Stripe/Auth ready)
- `/about` Recruiter-friendly portfolio page
